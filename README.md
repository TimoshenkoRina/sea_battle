Кроссплатформенная игра «Морской бой» на Python с графическим интерфейсом на Tkinter и компьютером‑соперником

## Возможности

- Поле 10×10 и классический набор кораблей: `4, 3, 3, 2, 2, 2, 1, 1, 1, 1`
- Автоматическая или ручная расстановка кораблей игрока на выбор
- После попадания компьютер добивает корабль, а не стреляет случайно
- Отмена последнего выстрела игрока
- Визуализация: две доски, подсветка попаданий, промахов, потопленных кораблей и зоны вокруг них
- Отображение списка оставшихся кораблей противника

## Правила

- У каждого игрока есть поле 10×10 с кораблями, которые противник не видит
- Ходы по очереди: вы кликаете по клеткам поля противника, компьютер стреляет по вашему полю
- Попадания и промахи отображаются цветом, вокруг потопленного корабля так же выделаются клетки
- Побеждает тот, кто первым потопит все корабли соперника

## Управление

- При запуске игра спрашивает, хотите ли вы вручную расставить корабли
  - Да:
    - кликайте по своему полю, чтобы ставить корабли;
    - корабли расставляются в порядке убывания, начиная с самого длинного;
    - клавиша `R` - смена ориентации расстановки (горизонтально/вертикально);
    - после расстановки всех кораблей начинается игра
  - Нет:
    - корабли расставляются автоматически, сразу можно стрелять по полю противника

- Левая доска — ваше поле
- Правая доска — поле противника
- Кнопка "Отменить ход" - откатывает последний выстрел игрока
- Кнопка "Новая игра»" - полностью сбрасывает состояние и запускает новую партию

## Внутренняя структура

- `gui.py` - графический интерфейс на Tkinter: главное окно, две доски, кнопки, статусы, обработка кликов
- `game.py` - игровая логика: создание полей, расстановка кораблей, выстрелы, ИИ, история ходов, отмена
- `models.py` - вспомогательные структуры данных: стек, очередь, отсортированные списки координат, бинарное дерево поиска
